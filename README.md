# homebrew-bb

Homebrew tap for [bb](https://github.com/azyu/bb-cli) — Bitbucket Cloud CLI.

## Install

```bash
brew install azyu/tap/bb
```

Or tap first:

```bash
brew tap azyu/tap
brew install bb
```

## Upgrade

```bash
brew upgrade bb
```

## Uninstall

```bash
brew uninstall bb
brew untap azyu/tap
```
